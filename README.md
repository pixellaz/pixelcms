PIXEL CMS
=============

A CMS that built on top of Django and MongoDB. This is a very adolescent CMS but promising a highly scaleable and easy to configure and maintenance.

VERSIONS
-------
### Version: 0.1 beta
- Basic facilities
- Admin CP
- FlatPages

Prequisites
-------
- Python 2.6+
- Django 1.2+
- MongoDB
- MongoEngine
- PyMongo
